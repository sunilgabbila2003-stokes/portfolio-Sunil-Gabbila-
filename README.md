## 📌 Overview

This is my personal developer portfolio — a fully hand-crafted, responsive single-page application showcasing my skills, projects, certifications, and contact information. Designed with a dark, tech-forward aesthetic inspired by modern developer tooling UIs.

---

## ✨ Features

- **Animated Particle Network** — Interactive canvas background with mouse-reactive connections
- **Scroll-Reveal Animations** — Elements animate into view using the Intersection Observer API
- **Smooth Counter Animation** — Hero stats count up on page load for visual impact
- **Active Nav Highlighting** — Navigation links dynamically highlight based on scroll position
- **Responsive Design** — Fully mobile-optimized with adaptive layouts at 768px breakpoint
- **Glowing Orb Effects** — CSS keyframe-animated radial gradient orbs in the hero section
- **Role-Based Sections** — Distinct sections: Hero, About, Skills, Projects, Testimonials, Education, Contact
- **No Dependencies** — Zero npm packages, zero build steps; pure browser-native code

## 🚀 Getting Started

### View Locally

No build process required. Just open the file:

```bash
git clone https://github.com/your-username/your-repo-name.git
cd your-repo-name
open index.html   # macOS
# or
start index.html  # Windows
```

### Deploy to GitHub Pages

1. Push `index.html` to the root of your repository
2. Go to **Settings → Pages**
3. Set Source to `Deploy from a branch` → `main` → `/ (root)`
4. Your site will be live at `https://your-username.github.io/your-repo-name`
